# Projet Lac Long

https://projet-lac-long.fly.dev/

## Dev

* Install [poetry](https://python-poetry.org/docs/#installation)

```
poetry shell
poetry install
uvicorn main:app --reload  
```

## Deploy

Install [fly](https://fly.io/docs/getting-started/installing-flyctl/)

```
fly deploy
```
